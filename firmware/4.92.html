STEP 1

Install HFW 4.92 from USB

  1. Connect the prepared matching HFW 4.92 USB.
  2. Open Settings → System Update.
  3. Select Update via Storage Media.
  4. Complete the update and return to this page.
STEP 2

Prepare the PS3 browser

  1. Open Browser → Tools.
  2. Set Confirm Browser Close to Off.
  3. Delete Cookies, Search History, Cache and Authentication Information.
  4. Close the browser completely.
  5. Open this firmware page again.
